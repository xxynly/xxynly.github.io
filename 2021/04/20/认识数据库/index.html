    第二关 认识数据库

    关系型数据库和非关系型数据库

    关系型数据库

    • 采用了关系模型来组织数据的数据库
    • 以行和列的形式存储数据
    • 行和列被称为表
    • 一组表组成数据库

    关系型数据库强调ACID规则

    • 原子性（Atomicity）
    • 一致性（Consistency）
    • 隔离性（Isolation）
    • 持久性（Durability）

    非关系型数据库

    • 即NoSQL（not only SQL），非关联型的
    • 强调Key-Value的方式存储和文档数据库的优点
    • 不遵循ACID原则

    数据库分类

    常用的关系型数据库

    • Oracle
    • Microsoft SQL Server
    • MySQL
    • SQLite
    • DB2
    • Microsoft Access

    常用的非关系型数据库

    • Redis
    • MongoDB
    • Memcached

    数据类型

    整数

    类型 含义
    tinyint(m) 1个字节 范围(-128~127)
    smallint(m) 2个字节 范围(-32768~32767)
    mediumint(m) 3个字节 范围(-8388608~8388607)
    int(m) 4个字节 范围(-2147483648~2147483647)
    bigint(m) 8个字节 范围(+-9.22*10的18次方)

    取值范围如果加了unsigned，则最大值翻倍

    浮点数

    类型 含义
    float(m,d) 单精度浮点型 8位精度(4字节) m总个数，d小数位
    double(m,d) 双精度浮点型 16位精度(8字节) m总个数，d小数位

    字符串

    类型 含义
    CHAR 定长字符串
    VARCHAR 变长字符串
    TINYBLOB 不超过 255 个字符的二进制字符串
    TINYTEXT 短文本字符串
    BLOB 二进制形式的长文本数据
    BLOB 长文本数据
    MEDIUMBLOB 二进制形式的中等长度文本数据
    MEDIUMTEXT 中等长度文本数据
    LONGBLOB 二进制形式的极大文本数据
    LONGTEXT 极大文本数据

    日期和时间

    类型 格式
    DATE YYYY-MM-DD
    TIME HH:MM:SS
    YEAR YYYY
    DATETIME YYYY-MM-DD HH:MM:SS
    TIMESTAMP YYYYMMDD HHMMSS

    定点数

    类型 含义
    decimal(m,d) 参数m<65 是总个数，d<30且 d<m 是小数位

    其他数据类型

    类型 含义
    BINARY 类似Char的二进制存储，特点是插入定长不足补0
    VARBINARY 类似VarChar的变长二进制存储，特点是定长不补0
    ENUM 最大可达65535个不同的枚举值
    SET 最大可达64个不同的值
    Geometry 经纬度 not null
    Point 简单点
    LineString 简单线
    MultiLineString 多线
    Polygon 简单面
    GeometryCollection 任何几何集合

    数据类型的属性

    MySQL关键字 含义
    NULL 数据列可包含NULL值
    NOT NULL 数据列不允许包含NULL值
    DEFAULT 默认值
    PRIMARY KEY 主键
    AUTO_INCREMENT 自动递增，适用于整数类型
    UNSIGNED 无符号
    character_set_name 指定一个字符集
